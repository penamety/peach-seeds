Additional Information:

The Morse-code heard in voice 1 at the start of the tune spells out the names of the games' creators. The decoded Morse-code is below. The (*) denotes where the tone of the Morse-code drops. The spelling is slightly incorrect, letter 'i' always translates as an 'e'. "I wrote that morse code program myself. Nobody touched it but me. It was written to handle any text. And I spelt the text right! So - if you're right - the problem must have been the software. I'm assuming
my morsecode book was right of course ;)" (MG)
